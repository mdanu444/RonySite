# Visa Management System - Complete Implementation Guide

## 🚀 System Overview

এই সিস্টেম একটি সম্পূর্ণ ভিসা ম্যানেজমেন্ট প্ল্যাটফর্ম যা ব্যবহারকারী লগইন, রেজিস্ট্রেশন, প্রোফাইল ম্যানেজমেন্ট এবং ভিসা CRUD অপারেশন সহ একটি ড্যাশবোর্ড প্রদান করে।

### প্রধান বৈশিষ্ট্য:
- ✅ ব্যবহারকারী রেজিস্ট্রেশন এবং লগইন
- ✅ ব্যবহারকারী প্রোফাইল সিস্টেম
- ✅ ভিসা CRUD (Create, Read, Update, Delete) সিস্টেম
- ✅ ড্যাশবোর্ড এবং রিপোর্টিং
- ✅ ভিসা সার্চ ফাংশনালিটি
- ✅ প্রোফাইল ফটো আপলোড
- ✅ সুন্দর রেসপন্সিভ ইন্টারফেস

---

## 📋 ইনস্টলেশন এবং সেটআপ

### ১. ডাটাবেস মাইগ্রেশন (ইতিমধ্যে সম্পন্ন)

ডাটাবেসে নতুন ফিল্ড যুক্ত করা হয়েছে:

**Users Table নতুন কলাম:**
- `phone_number` - ফোন নম্বর
- `date_of_birth` - জন্মতারিখ
- `gender` - লিঙ্গ (Male, Female, Other)
- `address` - ঠিকানা
- `profile_photo_path` - প্রোফাইল ছবি
- `bio` - বায়োগ্রাফি

**Visas Table (ইতিমধ্যে বিদ্যমান):**
- সমস্ত প্রয়োজনীয় ফিল্ড উপলব্ধ

## 🔐 লগইন এবং রেজিস্ট্রেশন

### নতুন ব্যবহারকারী রেজিস্ট্রেশন

```
URL: /register
Method: GET/POST
```

**রেজিস্ট্রেশন ফিল্ড:**
- Full Name (আবশ্যক)
- Email Address (আবশ্যক, ইউনিক)
- Password (আবশ্যক, কনফার্মেশন সহ)
- Phone Number (ঐচ্ছিক)
- Date of Birth (ঐচ্ছিক)
- Gender (ঐচ্ছিক)
- Address (ঐচ্ছিক)

### ব্যবহারকারী লগইন

```
URL: /login
Method: GET/POST
```

**লগইন ফিল্ড:**
- Email Address (আবশ্যক)
- Password (আবশ্যক)
- Remember Me (ঐচ্ছিক)

## 👤 ব্যবহারকারী প্রোফাইল সিস্টেম

### প্রোফাইল ভিউ (শুধু পড়া)

```
URL: /profile
Method: GET
Status: প্রয়োজনীয় সাইন-ইন
```

প্রোফাইল পেজে নিম্নলিখিত তথ্য প্রদর্শিত হয়:
- প্রোফাইল ছবি
- পূর্ণ নাম
- ইমেইল
- ফোন নম্বর
- জন্মতারিখ
- লিঙ্গ
- ঠিকানা
- বায়োগ্রাফি
- সদস্য হওয়ার তারিখ

### প্রোফাইল এডিট

```
URL: /profile/edit
Method: GET
Status: প্রয়োজনীয় সাইন-ইন
```

**এডিট করা যায় এমন ফিল্ড:**
- Full Name
- Email Address
- Phone Number
- Date of Birth
- Gender
- Address
- Bio
- Profile Photo (ছবি আপলোড)

**আপলোড করা ছবির জন্য:**
- সর্বোচ্চ সাইজ: 2MB
- সমর্থিত ফরম্যাট: JPG, PNG, GIF
- সংরক্ষণ স্থান: `storage/app/public/profile-photos`

## 📊 ড্যাশবোর্ড

### ড্যাশবোর্ড ভিউ

```
URL: /dashboard
Method: GET
Status: প্রয়োজনীয় সাইন-ইন
```

**ড্যাশবোর্ড উপাদান:**

1. **পরিসংখ্যান কার্ড:**
   - মোট ভিসা
   - সক্রিয় ভিসা
   - পেন্ডিং ভিসা
   - মেয়াদ উত্তীর্ণ ভিসা
   - মোট ব্যবহারকারী

2. **সাম্প্রতিক ভিসা তালিকা:**
   - সর্বশেষ ৫টি ভিসা
   - ভিসা নম্বর এবং স্ট্যাটাস সহ

3. **সাম্প্রতিক ব্যবহারকারী:**
   - সর্বশেষ ৫ জন ব্যবহারকারী
   - নাম এবং ইমেইল সহ

4. **দ্রুত কর্ম:**
   - নতুন ভিসা যোগ করুন
   - সমস্ত ভিসা দেখুন
   - প্রোফাইল এডিট করুন

## 🎫 ভিসা ম্যানেজমেন্ট সিস্টেম

### ভিসা তালিকা (সূচকীকরণ)

```
URL: /visas
Method: GET
Status: প্রয়োজনীয় সাইন-ইন
```

বৈশিষ্ট্য:
- সমস্ত ভিসা তালিকা পৃষ্ঠায় দেখুন
- তালিকায় ১০ ভিসা প্রতি পৃষ্ঠা
- প্রতিটি ভিসার জন্য পদক্ষেপ বোতাম: দেখুন, এডিট, মুছুন

---

### নতুন ভিসা যোগ করুন

```
URL: /visas/create
Method: GET/POST
Status: প্রয়োজনীয় সাইন-ইন
```

**ভিসা তথ্য:**
- Visa Number (আবশ্যক, ইউনিক)
- Visa Type (আবশ্যক) - যেমন টুরিস্ট, বিজনেস, স্টুডেন্ট
- Visa Status (আবশ্যক) - Active, Inactive, Pending, Expired
- Visa Validity Date (আবশ্যক)
- Purpose of Visit (আবশ্যক)

**ব্যক্তিগত তথ্য:**
- First Name (আবশ্যক)
- Surname (ঐচ্ছিক)
- Date of Birth (আবশ্যক)
- Citizenship (আবশ্যক)
- Passport Number (আবশ্যক)
- Photo (ঐচ্ছিক) - সর্বোচ্চ 2MB

---

### ভিসা বিস্তারিত দেখুন

```
URL: /visas/{id}
Method: GET
Status: প্রয়োজনীয় সাইন-ইন
```

বিস্তারিত পৃষ্ঠায় দেখা যায়:
- ভিসার ছবি
- সমস্ত ভিসা এবং ব্যক্তিগত তথ্য
- এডিট এবং মুছুন বোতাম

---

### ভিসা এডিট করুন

```
URL: /visas/{id}/edit
Method: GET/PUT
Status: প্রয়োজনীয় সাইন-ইন
```

সৃষ্টির মতো একই ফিল্ড এডিট করা যায়।

---

### ভিসা মুছুন

```
URL: /visas/{id}
Method: DELETE
Status: প্রয়োজনীয় সাইন-ইন
```

নিশ্চিতকরণ সহ ভিসা মুছে দিন।

---

### ভিসা সার্চ

```
URL: /visas/search
Method: GET
Status: প্রয়োজনীয় সাইন-ইন
```

**সার্চ প্যারামিটার:**
- `query` - ভিসা নম্বর, নাম বা উপনাম দ্বারা অনুসন্ধান

উদাহরণ:
```
GET /visas/search?query=VISA123
GET /visas/search?query=John
```

## 🛣️ সমস্ত API রুট

### পাবলিক রুট
```
GET  /                           → স্বাগত পৃষ্ঠা
POST /verify                     → ভিসা যাচাই (API)
GET  /register                   → রেজিস্ট্রেশন ফর্ম
POST /register                   → রেজিস্ট্রেশন সাবমিট
GET  /login                      → লগইন ফর্ম
POST /login                      → লগইন সাবমিট
```

### লগইন করা ব্যবহারকারীর রুট
```
GET  /dashboard                  → ড্যাশবোর্ড
POST /logout                     → লগআউট

# প্রোফাইল
GET  /profile                    → প্রোফাইল ভিউ
GET  /profile/edit               → প্রোফাইল এডিট
PUT  /profile                    → প্রোফাইল আপডেট

# ভিসা
GET    /visas                    → ভিসা তালিকা
GET    /visas/create             → ভিসা তৈরি ফর্ম
POST   /visas                    → ভিসা সংরক্ষণ
GET    /visas/{id}               → ভিসা বিস্তারিত
GET    /visas/{id}/edit          → ভিসা এডিট ফর্ম
PUT    /visas/{id}               → ভিসা আপডেট
DELETE /visas/{id}               → ভিসা মুছুন
GET    /visas/search             → ভিসা অনুসন্ধান
```

## 🔧 নিরাপত্তা বৈশিষ্ট্য

1. **পাসওয়ার্ড হ্যাশিং:** ব্যবহারকারী পাসওয়ার্ড ডাটাবেসে হ্যাশ করা হয়
2. **CSRF সুরক্ষা:** সমস্ত ফর্মে CSRF টোকেন
3. **প্রমাণীকরণ:** মিডলওয়্যার দ্বারা সুরক্ষিত রুটস
4. **যাচাইকরণ:** সমস্ত ইনপুট যাচাই করা হয়
5. **ইউনিক সীমাবদ্ধতা:** ইমেইল এবং ভিসা নম্বর অনন্য
6. **ফাইল আপলোড:** ইমেজ সাইজ এবং ধরন পরীক্ষা করা হয়

## 📧 পাবলিক ভিসা যাচাই API

পাবলিক ভিসা যাচাইকরণের জন্য কোনও সাইন-ইন প্রয়োজন নেই:

```bash
curl -X POST http://localhost:8000/verify \
  -H "Content-Type: application/json" \
  -d '{"visa_number":"VISA123458"}'
```

**প্রতিক্রিয়া উদাহরণ:**
```json
{
  "id": 1,
  "visa_number": "VISA123458",
  "firstname": "John",
  "surname": "Doe",
  "citizenship": "USA",
  "visa_status": "Active",
  "visa_validity": "2023-12-31",
  ...
}
```
